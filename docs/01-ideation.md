# 01. チームでアイデアを作る

3名程度のチームで、LINEミニアプリとして作りたい小さなWebアプリを考えます。

## 目標

GitHub Copilotに渡せる **アイデア引き継ぎパッケージ** を作ります。

```text
work/
  idea.md
  ui-sketch.png
```

画像ファイル名は `ui-sketch.jpg` や `wireframe.png` でも構いません。`work/idea.md` の「UIスケッチ」に実際のファイル名を書いてください。

## アイデアの絞り方

70分で完成に近づけるため、以下を意識します。

- 1〜3画面で終わる
- 1つの楽しい体験、または1つの課題解決に絞る
- ログイン、決済、データベース、外部APIを必須にしない
- LINEらしさはMockプロフィール、結果カード、シェア風UIなどで表現する

## おすすめテーマ

- 診断・おすすめアプリ
- イベントチェックインカード
- スタンプカード
- クイズ・投票
- プロフィールを使ったパーソナライズ体験

## Copilotへの相談例

Copilot Chatで次のように相談できます。

```text
3名チームでLINEミニアプリのアイデアを考えています。
70分でAIが実装できる、1〜3画面の小さなWebアプリ案を5つ出してください。
イベント参加者が使って楽しいものにしたいです。
```

アイデアが決まったら、`work/idea.md` を埋めます。
